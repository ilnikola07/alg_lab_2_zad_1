# alg_lab_2_zad_1
Это 1 задание лаборатоной работы №2. Приложение написано на языке C# и с помощью Windows Forms реализует графический интерфейс для общения с пользователем. 
Оно проводит анализ введённого пользователем числа, в результате чего находит максимальную и минимальную цифры в числе, а также подсчитывает количество каждой цифры в данном числе.
